Контекстна панель інструментів - це плаваюча панель дій, яка з'являється на кожній сторінці ваших документів. Він дозволяє читачам миттєво копіювати вміст сторінки, відкривати його в асистенті штучного інтелекту або додавати ваші документи як MCP-сервер - не залишаючи сторінки.

Налаштуйте його в `docs.json` в розділі `contextual.options`:

```json docs.json
{
  "contextual": {
    "options": [ "copy", "view", "claude", "chatgpt", "mcp", "cursor", "vscode"]
  }
}
```

## Доступні варіанти

### Дії з контентом

| Варіант     | Опис                                                       |
| ----------- | ---------------------------------------------------------- |
| Копіюй.     | Копіює сторінку як Markdown в буфер обміну                 |
| "перегляд". | Відкриває сиру версію сторінки Markdown (URL-адреса `.md`) |

### Асистенти зі штучним інтелектом

| Варіант        | Опис                                      |
| -------------- | ----------------------------------------- |
| "Клод          | Відкриває сторінку в Клоді                |
| `chatgpt`      | Відкриває сторінку в ChatGPT              |
| "розгубленість | Відкриває сторінку в розгубленості        |
| "Грок          | Відкриває сторінку в Grok                 |
| "aistudio      | Відкривається сторінка в Google AI Studio |

### Інтеграції з редакторами

| Варіант      | Опис                                         |
| ------------ | -------------------------------------------- |
| "курсор      | Додає сторінку до контексту курсору          |
| `vscode`     | Додає сторінку до VS Code Copilot            |
| "віндсерфінг | Додає сторінку до каскаду Windsurf's Cascade |
| "Девін".     | Відкриває сторінку в Девіні                  |

### MCP

| Варіант   | Опис                                                                                         |
| --------- | -------------------------------------------------------------------------------------------- |
| mcp.      | Показує інструкції з використання ваших документів як MCP-сервера                            |
| `add-mcp` | Надає підказку в один клік, щоб додати ваш docs MCP-сервер до інструменту штучного інтелекту |

<Tip>
  Опції `mcp` і `add-mcp` особливо потужні - вони дозволяють розробникам підключити весь ваш сайт з документацією до будь-якого асистента ШІ, сумісного з MCP, щоб він міг безпосередньо відповідати на запитання про ваш продукт.
</Tip>

## Користувацькі опції

Ви можете додавати власні записи на панель інструментів з назвою, описом, піктограмою та посиланням. href\` підтримує динамічні заповнювачі:

| Заповнювач | Вирішує                            |
| ---------- | ---------------------------------- |
| $page      | Повна URL-адреса поточної сторінки |
| $path      | Шлях до поточної сторінки          |
| $mcp       | URL-адреса вашого MCP-сервера docs |

```json docs.json
{
  "contextual": {
    "options": {
      "copy",
      "claude",
      {
        "title": "Відкрите питання",
        "description": "Повідомити про проблему з цією сторінкою",
        "icon": "github",
        "href": "https://github.com/your-org/docs/issues/new?title=Issue+on+$path"
      }
    ]
  }
}
```

<Note>
  Контекстну панель інструментів видно лише у попередньому та продуктивному розгортанні, але не у локальному попередньому перегляді `mint dev`.
</Note>

## Поєднання з видимістю

Опції панелі інструментів "копіювати" і "переглянути" відкривають безпосередньо вміст розмітки вашої сторінки. Якщо ви використовуєте компонент [`Visibility`] (/agent-ready/visibility), блоки `for="agents"` будуть включені в цю розмітку, надаючи ШІ-помічникам структурований контекст, який не бачать читачі-люди.
