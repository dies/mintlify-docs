Налаштуйте Claude Code, щоб допомогти вам писати і підтримувати документацію Mintlify.

## Передумови

* Активна підписка Claude (Pro, Max або доступ до API)

## Налаштування

<Steps>
<Step title="Встановіть код Клода">
  ```bash
  npm install -g @anthropic-ai/claude-code
  ```
</Step>

<Step title="Встановіть навичку Mintlify">
  У вашому каталозі docs запустіть:

  ```bash
  npx навички додати https://mintlify.com/docs
  ```

  Тут ви знайдете посилання на компоненти Claude Code Mintlify, стандарти написання,
  та інструкції щодо робочого процесу.
</Step>

<Step title="Налаштуйте інструкції проекту">
  Відредагуйте `AGENTS.md` у корені вашого проекту, щоб додати специфічну для проекту термінологію, налаштування стилю
  та межі вмісту.
</Step>

<Step title="Почніть писати з Клод Кодом">
  ```bash
  Клод
  ```
</Step>
</Steps>
