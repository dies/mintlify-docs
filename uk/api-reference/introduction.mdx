## Ласкаво просимо.

Існує два способи побудови документації до API: [OpenAPI](https://mintlify.com/docs/api-playground/openapi/setup) та [MDX компоненти](https://mintlify.com/docs/api-playground/mdx/configuration). Для стартового набору ми використовуємо наступну специфікацію OpenAPI.

## Аутентифікація

Всі кінцеві точки API аутентифікуються за допомогою токенів Bearer, які беруться з файлу специфікації.

```json
"security": [
  {
    "bearerAuth": [
  }
]
```
