## Вбудований код

Щоб позначити `слово` або `фразу` як код, візьміть їх у лапки (\`).

## Блоки коду

Використовуйте [огороджені блоки коду](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks), оточивши код трьома бектіками, і слідуйте за провідними тіками з мовою програмування вашого фрагмента, щоб отримати підсвічування синтаксису. За бажанням, ви також можете написати назву вашого коду після мови програмування.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```


## Групи кодів

Щоб відобразити той самий фрагмент кількома мовами у вигляді вкладок, використовуйте компонент `CodeGroup`. Дивіться [Компоненти](/essentials/components#code-groups) для використання.
