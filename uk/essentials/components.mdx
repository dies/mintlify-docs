Компоненти Mintlify дозволяють створювати багаті, інтерактивні сторінки документації за допомогою MDX. Ось деякі з найпоширеніших.

Повний список можна знайти в [огляді компонентів] (https://mintlify.com/docs/components) в документації Mintlify.

## Вигуки

Використовуйте вигуки, щоб привернути увагу до важливої інформації.

<Note>**Примітка** містить загальну інформацію, яка допоможе читачеві.</Note>

<Tip>Підказка\*\* вказує на корисний спосіб скорочення або найкращу практику.</Tip>

<Warning>Попередження\*\* вказує на те, що може спричинити проблеми.</Warning>

<Info>Вигук **Інформація** надає додатковий контекст.</Info>

```mdx
<Note>**Примітка** містить загальну інформацію, яка допоможе читачеві.</Note>

<Tip>Порада** вказує на корисний ярлик або найкращу практику.</Tip>

<Warning>Попередження" вказує на те, що може спричинити проблеми.</Warning>

<Info>Вигук **Інформація** надає додатковий контекст.</Info>
```

## Сходинки

Кроки чітко представляють пронумеровані процедури.

<Steps>
  <Step title="Встановіть пакунок">
    ```bash
    npm install my-package
    ```
  </Step>
  <Step title="Ініціалізувати">
    Запустіть команду init у корені вашого проекту.
    ```bash
    npx my-package init
    ```
  </Step>
  <Step title="Ти готова.">
    Почніть використовувати пакет у своєму коді.
  </Step>
</Steps>

````mdx
<Steps>
  <Step title="Install the package">
    ```bash
    npm install my-package
    ```
  </Step>
  <Step title="Initialize">
    Запустіть команду init у корені вашого проекту.
  </Step>
  <Step title="You're ready">
    Почніть використовувати пакунок у вашому коді.
  </Step>
</Steps>
````

## Вкладки

Вкладки дозволяють читачам перемикатися між альтернативним контентом - різними мовами, середовищами або платформами.

<Tabs>
  <Tab title="проміле">
    ```bash
    npm install my-package
    ```
  </Tab>
  <Tab title="пряжа">
    ```bash
    пряжа додати мій пакет
    ```
  </Tab>
  <Tab title="проміле">
    ```bash
    pnpm add my-package
    ```
  </Tab>
</Tabs>

````mdx
<Tabs>
  <Tab title="npm">
    ```bash
    npm install my-package
    ```
  </Tab>
  <Tab title="yarn">
    ```bash
    yarn add my-package
    ```
  </Tab>
  <Tab title="pnpm">
    ```bash
    pnpm add my-package
    ```
  </Tab>
</Tabs>
````

## Акордеони

Акордеони згортають вміст за перемикачем - корисно для поширених запитань або необов'язкових деталей.

<AccordionGroup>
  <Accordion title="Що робить цей продукт?">
    Коротка відповідь на перше питання. Будь лаконічним.
  </Accordion>
  <Accordion title="Як розраховується ціна?">
    Ще одна відповідь. Використовуйте стільки місця, скільки вам потрібно.
  </Accordion>
</AccordionGroup>

```mdx
<AccordionGroup>
  <Accordion title="What does this product do?">
    Коротка відповідь на перше питання.
  </Accordion>
  <Accordion title="How is pricing calculated?">
    Ще одна відповідь тут.
  </Accordion>
</AccordionGroup>
```

## Групи кодів

`CodeGroup` відображає кілька фрагментів коду у вигляді вкладок - ідеально підходить для показу одного і того ж прикладу кількома мовами.

<CodeGroup>
  ```typescript TypeScript
  const response = await fetch('https://api.example.com/users');
  const users = await response.json();
  ```

  ```python Python
  import requests
  response = requests.get('https://api.example.com/users')
  users = response.json()
  ```

  ```bash curl
  curl https://api.example.com/users
  ```
</CodeGroup>

````mdx
<CodeGroup>
  ```typescript TypeScript
  const response = await fetch('https://api.example.com/users');
  ```

  ```python Python
  response = requests.get('https://api.example.com/users')
  ```
</CodeGroup>
````
