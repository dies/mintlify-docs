## Зображення

### Використовуючи уцінку

Синтаксис [markdown синтаксис](https://www.markdownguide.org/basic-syntax/#images) дозволяє додавати зображення за допомогою наступного коду

```md
![title](/path/image.jpg)
```

Зверніть увагу, що розмір файлу зображення не повинен перевищувати 5 МБ. В іншому випадку ми рекомендуємо хостинг на таких сервісах, як [Cloudinary](https://cloudinary.com/) або [S3](https://aws.amazon.com/s3/). Потім ви можете використовувати цю URL-адресу для вбудовування.

### Використання вбудовувань

Щоб отримати більше можливостей для налаштування зображень, ви також можете використовувати [embeds](/writing-content/embed) для додавання зображень

```html
<img height="200" src="/path/image.jpg" />
```

### iFrames

Завантажує ще одну HTML-сторінку в документі. Найчастіше використовується для вбудовування відео.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
