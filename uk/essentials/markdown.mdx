## Назви

Найкраще використовувати для заголовків розділів.

```md
## Назви
```

### Субтитри

Найкраще використовувати для заголовків підрозділів.

```md
### Субтитри
```

<Tip>

Кожен **назва** і **підзаголовок** створює якір, а також відображається у змісті праворуч.

</Tip>

## Форматування тексту

Ми підтримуємо більшість форматів націнок. Просто додайте `**`, `_` або `~` навколо тексту, щоб відформатувати його.

| Стиль      | Як його написати | Результат           |
| ---------- | ---------------- | ------------------- |
| Сміливіше. | Жирним шрифтом.  | \*\*жирним шрифтом. |
| Курсив     | Курсив.          | *italic*            |
| Закреслено | Закреслено.      | ~закреслено.        |

Ви можете поєднувати їх. Наприклад, напишіть `**_bold and italic_**`, щоб отримати ***bold and italic*** текст.

Для написання надрядкового та підрядкового тексту потрібно використовувати HTML. Тобто, додайте `\<sup>` або `\<sub>` навколо вашого тексту.

| Розмір тексту      | Як його написати                           | Результат                   |
| ------------------ | ------------------------------------------ | --------------------------- |
| Надрядковий індекс | <sup>`superscript`</sup>(надрядковий знак) | <sup>надрядковий знак</sup> |
| Підпис             | <sub>підписка</sub>                        | <sub>підпис</sub>           |

## Посилання на сторінки

Ви можете додати посилання, обернувши текст у `[]()`. Ви маєте написати `[посилання на google](https://google.com)` до [посилання на google](https://google.com).

Посилання на сторінки у ваших документах мають бути кореневими. В основному, ви повинні вказати весь шлях до теки. Наприклад, `[посилання на текст](/writing-content/text)` посилається на сторінку "Текст" у нашому розділі компонентів.

Відносні посилання типу `[посилання на текст](../text)` відкриватимуться повільніше, оскільки ми не можемо оптимізувати їх так само легко.

## Блокчейн

### Однолінійний

Щоб створити лапки, додайте `>` перед абзацом.

> Дороті пішла за нею по багатьох прекрасних кімнатах її замку.

```md
> Дороті пішла за нею по багатьох прекрасних кімнатах її замку.
```

### Багаторядковий

> Дороті пішла за нею по багатьох прекрасних кімнатах її замку.
>
> Відьма наказала їй вимити каструлі та чайники, підмести підлогу і підтримувати вогонь дровами.

```md
> Дороті ходила за нею по багатьох прекрасних кімнатах її замку.
>
> Відьма наказала їй мити каструлі та чайники, підмітати підлогу і підтримувати вогонь дровами.
```

### LaTeX

Mintlify підтримує [LaTeX] (https://www.latex-project.org) через компонент Latex.

<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>

```md
<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>
```
