## Почніть у три кроки

Запустіть свій сайт документації локально і зробіть перші налаштування.

### Крок 1: Налаштуйте локальне середовище

<AccordionGroup>
  <Accordion icon="копія" title="Клонуйте документи локально">
    Під час адаптації ви створили репозиторій на GitHub зі своїми документами, якщо у вас його ще не було. Ви можете знайти посилання на цей репозиторій у вашій [інформаційній панелі] (https://dashboard.mintlify.com).

    Щоб клонувати сховище локально, щоб ви могли вносити зміни до своїх документів і попередньо переглядати їх, дотримуйтесь інструкції [Клонування сховища] (https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository) у документації GitHub.
  </Accordion>
  <Accordion icon="прямокутник-термінал" title="Запустіть сервер попереднього перегляду">
    1. Встановіть Mintlify CLI: `npm i -g mint`.
    2. Перейдіть до каталогу docs і запустіть: `mint dev`.
    3. Відкрийте `http://localhost:3000`, щоб побачити ваші документи наживо!

    <Tip>Попередній перегляд оновлюється автоматично під час редагування файлів.</Tip>
  </Accordion>
</AccordionGroup>

### Крок 2: Впровадьте зміни

<AccordionGroup>
  <Accordion icon="github" title="Встановіть наш додаток GitHub">
    Встановіть додаток Mintlify GitHub з вашої [інформаційної панелі] (https://dashboard.mintlify.com/settings/organization/github-app).

    Наш додаток GitHub автоматично розгортає ваші зміни на вашому сайті, тому вам не потрібно керувати розгортанням самостійно.
  </Accordion>
<Accordion icon="палітра" title="Оновіть назву та кольори вашого сайту">
    Для початку давайте оновимо назву та кольори вашого сайту docs.

    1. Відкрийте `docs.json` у вашому редакторі.
    2. Змініть поле `"name"` на назву вашого проекту.
    3. Оновіть "кольори", щоб вони відповідали вашому бренду.
    4. Зберігайте та миттєво переглядайте зміни на `http://localhost:3000`.

    <Tip>Спробуйте змінити основний колір, щоб одразу побачити різницю!</Tip>
</Accordion>
</AccordionGroup>

### Крок 3: Починаємо роботу

<Accordion icon="ракета" title="Публікуйте свої документи">
  1. Візьміть на себе зобов'язання і просувайте свої зміни.
  2. Ваші документи оновляться і стануть доступними за мить!
</Accordion>
